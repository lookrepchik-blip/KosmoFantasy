# КосмоФентези Telegram Bot v7

Улучшенная версия Telegram-бота для фэнтези-лиги на базе Excel-файла `КосмоФентези_v4_BeteraPool_complete.xlsx`.

## Что нового в v7.1
- состав собирается 1 раз на регулярку и 1 раз на весь плей-офф
- замены применяются сразу
- регулярка: 1 замена за сезон
- плей-офф: до 2 замен после каждого тура (лимит обновляется каждый новый раунд)
- ввод очков переведён в режим **только администратор**
- добавлен `/transfers` для контроля замен
- админ выбирает участника, затем игрока, и вносит статистику кнопками
- подготовлен базовый модуль `/flashscore` как заготовка под подтверждаемый импорт
- улучшены тексты, меню и статусы

## Что умеет
- привязка Telegram-аккаунта к участнику
- базовый состав 1 раз на регулярку и 1 раз на весь плей-офф
- ограничение: только 1 полевой игрок из клуба, вратарь — исключение
- дедлайн и ручная блокировка составов
- замены с учётом этапа сезона
- ввод статистики по игрокам только администратором
- правила из Excel через `/rules`
- лидерборд
- админ-команды для раунда, стадии, дедлайна и лимитов состава
- экспорт в Excel, включая историю замен

## Быстрый старт
1. Создай бота через `@BotFather`
2. Скопируй `.env.example` в `.env`
3. Заполни `BOT_TOKEN` и `ADMIN_IDS`
4. Убедись, что рядом лежит Excel-файл `КосмоФентези_v4_BeteraPool_complete.xlsx`
5. Установи зависимости:
   ```bash
   pip install -r requirements.txt
   ```
6. Запусти:
   ```bash
   python app.py
   ```

## Команды пользователя
- `/start` — старт и привязка
- `/menu` — главное меню
- `/status` — статус текущего этапа и дедлайна
- `/me` — мой профиль
- `/lineup` — собрать базовый состав этапа
- `/transfer` — сделать замену
- `/mylineup` — посмотреть активный состав
- `/transfers` — посмотреть лимит и историю замен
- `/myscores` — мои записи
- `/leaderboard` — рейтинг
- `/rules` — правила
- `/help` — подсказка

## Команды админа
- `/admin`
- `/score`
- `/flashscore`
- `/setround <номер>`
- `/setstage <название>`
- `/deadline YYYY-MM-DD HH:MM`
- `/locklineup`
- `/unlocklineup`
- `/setlimits Вратарь=1,Защитник=2,Нападающий=2`
- `/export`

## Логика очков
- полевые: `голы + передачи + ручная поправка`
- вратарь:
  - шатаут = 3
  - 1 пропущенная = 2
  - 2 пропущенные = 1
  - 3+ пропущенных = 0
  - далее добавляется ручная поправка

## Экспорт
Бот создаёт или обновляет листы:
- `BotLineups`
- `BotScores`
- `BotLeaderboard`
- `BotUsers`
- `BotTransfers`

## Практические замечания
- текущая версия использует надёжный режим: очки заносит администратор
- лимиты по позициям можно менять через `.env` или командой `/setlimits`
- бот читает общий пул игроков с листа `BeteraPool`
